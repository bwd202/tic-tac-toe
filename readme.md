# How to play
- write your name in the input field [optional]
- press **start**
- click anywhere on the board to play
- press **reset** to clear the board

#### Rules
- first player to get 3 marks in a row, column or diagonal wins the game
- player 1 is 'X'
- player 2 is 'O'
